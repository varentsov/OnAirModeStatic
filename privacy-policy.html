Privacy Policy for On Air Mode

Last Updated: June 26, 2025

Overview

On Air Mode is a macOS application that automatically enables macOS Focus mode when audio input devices are being used by applications. This privacy policy explains how we handle your information.

Key Point: On Air Mode processes all data locally on your device and does not collect, store, or transmit any personal information to external servers.

Information We Collect

Audio Device Monitoring

  • Purpose: To detect when audio input devices are actively being used by applications
  • Data Processed: Real-time audio device activity status (active/inactive)
  • Storage: No audio data is accessed, recorded, stored, or transmitted
  • Local Processing: All device monitoring occurs locally using system hardware APIs

System Information

  • Purpose: To control macOS Focus mode through system integrations
  • Data Processed: Focus mode status changes via macOS Shortcuts
  • Storage: No system state data is stored permanently

User Preferences

  • Purpose: To remember your app settings and preferences
  • Data Stored:
    • Launch at login preference
    • Monitoring enabled/disabled state
  • Location: Stored locally in macOS UserDefaults

How We Use Your Information

  • Device Monitoring: Detect when audio input devices become active to trigger Focus mode
  • System Integration: Control Focus mode through macOS Shortcuts functionality
  • User Experience: Remember your preferences for app behavior

Data Sharing and Disclosure

  • No Data Sharing: We do not share, sell, or transmit any data to third parties
  • Local Processing: All functionality operates entirely on your local device
  • No Analytics: We do not collect analytics or usage statistics
  • No Remote Servers: The app does not communicate with external servers

Data Security

  • Sandboxed Environment: App runs in macOS App Sandbox for security
  • Local Storage: All data remains on your device
  • No Network Access: App does not require or use internet connectivity
  • System Permissions: Uses only necessary macOS permissions for functionality

Your Rights and Choices

Permission Control

  • Automation Permission: You can manage AppleScript/Shortcuts permissions in System Preferences > Security & Privacy > Automation

Data Deletion

  • Uninstall: Removing the app deletes all associated data
  • Reset Preferences: User preferences can be reset by removing the app and reinstalling

Children's Privacy

On Air Mode does not knowingly collect personal information from children under 13. The app's functionality does not involve personal data collection.

Changes to This Policy

We may update this privacy policy from time to time. We will notify users of any material changes by updating the "Last Updated" date at the top of this policy.

Third-Party Services

macOS Shortcuts Integration

  • Purpose: To control Focus mode functionality
  • Data: Commands sent to macOS Shortcuts app (on/off states)
  • Privacy: Subject to Apple's privacy policy for macOS system services

Technical Implementation

What We Monitor

  • Audio input device activity status through Core Audio hardware APIs
  • No audio content is accessed, recorded, or analyzed

What We Don't Collect

  • Audio recordings or content
  • Personal conversations or communications
  • User behavior patterns
  • Device identification information
  • Location data
  • Contact information
